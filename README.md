# **Static for quickmongo** 
Static quickmongo
#

# **Installation**
```
npm i quickmongo-static
```
# **Usage**

```js

const <Database> = require('quickmongo-static')

<Database>.on("ready", () => {
    console.log("Database connected!");
});

```
